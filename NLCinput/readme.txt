OpenNLCI installation instructions (quick):

- Install OpenNLC dependencies (see opennlc.net for more information).
- Open NLCIglobalDefs.hpp and configure all input folders.
- Open SHAREDglobalDefs.hpp and ensure that COMPILE_NLCI is uncommented.
- Open and build OpenNLCI.pro with QT creator.
- Ensure that all input files (including NLCrules.xml/GIArules.xml) are copied to NLCinput.
